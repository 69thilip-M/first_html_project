Framemaker interface

Generating HTML
           These are scripts for generating SGML hypertext from things like directory listings, etc. Also, for checking and correcting dubious HTML.
Server log analysis
           Analysing server logs requires first of all changing the numeric internet node numebrs into domain names. httpd-analyse.cis a program to do that. Feed the results through awk and grep of your choice!
